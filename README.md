# 🌐 CloudDeploy Platform (DevOps NGINX Demo)

## 🚀 Intro
This project is a simple **static web page demo** that can be deployed using **Nginx** (manually or via Docker).  
It is useful for understanding basic web hosting in a DevOps environment.

---

## 🧩 Features
- Static HTML landing page
- Nginx web server compatible
- Docker-ready deployment
- Simple DevOps learning project
